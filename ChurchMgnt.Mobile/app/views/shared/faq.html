Why the family member not showing in the offering page?
Offering page shows the family members if they are a taxpayer. So please go MemberCare -> Family -> Edit family then check Tax Payer option then save.
Why am I not able to edit/ delete/ create offering/ expense/ deposit for previous years?
Once you filed the tax, System will not allow you doing any action on tax filed years.
How to add new expesne type/ fund type/ offering type/ sub expense type/ event type?
Go to Left nav bar -> Master data -> select appropriate menu.
Why does report page not allowing to select more 12 months date range?
The system only supports 12 months date range in the report, If you want to more data, Please contact ChurchMgnt support team..
How to report an issue to ChurchMgnt support team?.
There is action link available from right side on page header close to logout action link.
